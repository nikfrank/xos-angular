here's a demo lesson on a super simple app I just wrote

there's a bug in the calculator! it works for - / and *, but adding is crazy!

what's happening is the "+" operator has a different meaning for numbers as for strings

in our add function, we have::

$scope.ans = $scope.num1 + $scope.num2;

which would be fine as long as num1 and num2 are both numbers

buuuut, if they're strings, the "+" will act as a string concatenator instead of adiition

and since we aren't telling angular what type to expect, it uses what it sees default from the input box (string)

the other operators work because they're only defined for numbers.

so there's two ways of fixing this:: from the view or the controller

we can either put a type="number" attribute on the input boxes (which I like better, as this also indicicates to the browser how he should render the input box)

or if we don't want that, we can parse the input into an integer in the add function
