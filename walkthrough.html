there's a bug in the calculator! it works for - / and *, but adding is crazy!

Solution

as you can see, the type is a string (not a number)

there's two ways to fix this. Either we can parseInt num1 and num2 into numbers in the controller using

$scope.ans = parseInt($scope.num1) + parseInt($scope.num2);

or, we can put a type="number" attribute on the input elements (this is better because it tells the browser how to render)
